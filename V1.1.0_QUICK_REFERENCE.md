## Under the Hood

When you save converted tags:

1. Component detects .cue file for track
2. Reads original file (detects encoding)
3. Updates only TITLE/PERFORMER/COMMENT fields
4. Keeps everything else exactly as is
5. Writes as UTF-8 (modern standard)
6. Creates .cue.bak backup first
7. Updates foobar2000 database

**Performance:** ~50ms for typical CUE file (no UI blocking)

## Supported Encodings

**Read:** UTF-8, GB2312, GBK, BIG5, and others (auto-detected)  
**Write:** UTF-8 with BOM
